## 题目描述
我们可以用2*1的小矩形横着或者竖着去覆盖更大的矩形。请问用n个2*1的小矩形无重叠地覆盖一个2*n的大矩形，总共有多少种方法？

### 问题分解

最后一块砖如何摆？横着摆是一种，竖着摆是一种。
f(n) = f(n-1) + f(n-2)

```javascript
function rectCover(number)
{
    var prev = 1, 
        next = 2, 
        i;
    
    if (number < 3) {
        return number;
    }
    for (i = 3; i <= number; i++) {
        next = prev + next;
        prev = next - prev;
    }
    return next;
}
```
