## 题目描述
输入一个链表，按链表值从尾到头的顺序返回一个ArrayList。

## JavaScript解法

### 数组的堆栈方法和倒置方法

```javascript
/*function ListNode(x){
    this.val = x;
    this.next = null;
}*/
function printListFromTailToHead(head)
{
    var stack = [];
    while (head) {
        stack.push(head.val);
        head = head.next;
    }
    return stack.reverse();
}
```
