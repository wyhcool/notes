## 题目描述
一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

## JavaScript解法

### 根据通项公式转换为循环解法
f(n) = f(n-1) + f(n-2);
f(2) = 2;
f(1) = 1;

```javascript
function jumpFloor(number)
{
    var jump1 = 1,
        jump2 = 2,
        jumps;
    
    if (number < 3) {
        jumps = number;
    } else {
        while (number >= 3) {
            jumps = jump1 + jump2;
            jump1 = jump2;
            jump2 = jumps;
            number--;
        }
    }
    return jumps;
}
```
