## 题目描述
输入一个链表，输出该链表中倒数第k个结点。

## JavaScript解法

### 空间换时间

建立顺序索引。

```javascript
/*function ListNode(x){
    this.val = x;
    this.next = null;
}*/
function FindKthToTail(head, k)
{
    var reference = [];
    
    while (head) {
        reference.push(head);
        head = head.next;
    }
    
    return reference[reference.length - k];
}
```
