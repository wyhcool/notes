## 题目描述
输入一个链表，反转链表后，输出新链表的表头。

## JavaScript解法

### 遍历拼接

注意操作顺序。

```javascript
/*function ListNode(x){
    this.val = x;
    this.next = null;
}*/
function ReverseList(pHead)
{
    // write code here
    var reverseHead = null,
        node = null;
    
    while (pHead) {
        node = pHead;
        pHead = pHead.next;
        if (reverseHead === null) {
            node.next = null;
            reverseHead = node;
        } else {
            node.next = reverseHead;
            reverseHead = node;
        }
    }
    
    return reverseHead;
}
```
