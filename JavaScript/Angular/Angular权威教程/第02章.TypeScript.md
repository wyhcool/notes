TypeScript 是 ES6 的超集，所有 ES6 代码都是完全有效且可编译的 TypeScript 代码。

TypeScript 相对于 ES5 有五大改善：
- 类型系统
- 类
- 注解
- 模块导入
- 语言工具包(比如，解构)
